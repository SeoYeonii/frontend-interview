## frontend-interview

이직 준비를 하면서 여러 자료들과 책을 보며 질문지를 리스트화 하던 중 <br />
개인적으로 더 관리하고 싶고 <br />
또 여러 깃허브 문서에서 많은 도움을 받았으니, 혹시 또 다른 개발자분들께 도움이 되지 않을까 싶어 <br />
프론트엔드 개발자를 위한 면접 문답지를 만들어보았습니다.
